05 / SQL logic

The dashboard is built from six business questions.

All sections map back to the original SQL queries, making the portfolio easy to verify for analyst and operations roles.

GROUP BYSUM / AVGNULLIFRANK window functionHAVING filterscommercial KPI logic

The project demonstrates SQL for practical business diagnostics rather than isolated syntax exercises.

-- Example logic
SELECT campaign_channel,
       SUM(clicks) AS total_clicks,
       SUM(conversions) AS total_conversions,
       ROUND(100.0 * SUM(conversions) / NULLIF(SUM(clicks), 0), 2) AS conversion_rate_pct
FROM customer_campaign_performance
GROUP BY campaign_channel
ORDER BY conversion_rate_pct DESC;
